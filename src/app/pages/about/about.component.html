lizt.co - checklists made easy

community driven website for creating and sharing checklists

Hello, my name is Kamil, and I've created this Open Source project to solve one of my problems.

I'm a software developer, and often I have to do some repetitive task that requires fulfilling many steps and none of them can be omitted. For example: if you want to publish an application on App Store, you have to create a description, make screenshots, prepare tags, publish Privacy Policy, change server URL to production, etc. These are tasks that are done once or twice a year by every iOS developer. Mentioned steps are easy, but there is a lot to cover. If you forget about some, you will end by making another build. That took time. The easiest solution for this is to create a humble checklist and check it before submitting the application.

This was the basic idea behind lizt.co website. But I made it more social. I want to share my checklists, and I think everyone can make something useful for others. So don't hesitate, play with lizt.co, publish own checklists or share them using a link.
